      Literals #

      A literal is a value that is specified directly in the source code.

      Ballerina supports the following literals.

      LiteralsExample
      Nil Literal(), null
      Boolean Literaltrue or false
      Integer Literal123, 0x1A
      Floating Point Literal456.78, 12.3f, 12.3d, 1.23e3, 0X1a.A
      String Literal"Hello, World!"
      Byte Array Literalbase16 `AB 12 34`
      base64 `A1z+`
      Nil Literal
      Nil literal uses the syntax () and null.
      Boolean Literal
      Boolean literal syntax is used to write Boolean values. | A boolean true is written as true and a boolean false is written as false.
      Integer Literal
      Integer values can be expressed in either common the Decimal system, known for its base 10 format, or the Hexadecimal system which uses the Base 16 format.
      Floating Point Literal
      Floating point values can be expressed in either Decimal (base 10) or Hexadecimal (base 16) systems | and come in two types, float and decimal, with slight differences in their representation using floating point literals.
      String Literal
      A string literal can be written by having the Unicode code points wrapped using two double quotes ".
      Byte Array Literal
      Byte Array Literal is a sequence of bytes represented in base16 or base64.

      Deciding the Numerical Literal’s type (Advanced) #

      A value written in a numeric literal always represents a specific type, which is determined by the literal itself. The type of a literal can be one of the basic types, such as int, float, or decimal.

      For example, the literal 10 represents the integer value 10, and its type is int. However, in some contexts, the same literal 10 can also represent a floating-point value 10.0 or a decimal value 10. Depending on the context, the compiler determines the appropriate type of the literal to be used.

       1import ballerina/io;
 2
 3const int TEN = 10;
 4
 5public function main() {
 6
 7  var a1 = 10;
 8  TEN a2 = 10;
 9  float a3 = 10;
10  decimal a4 = 10;
11  float|decimal a5 = 10;
12  var a6 = 10.0;
13
14  io:println(a1 is int);
15  io:println(a2 is TEN);
16  io:println(a3);
17  io:println(a4);
18  io:println(a5 is float);
19  io:println(a6 is float);
20}
      Code Breakdown
      a1’s type is int.
      a2’s type is singleton integer 10.
      You can use integer literal 10 as a floating point literal 10.0.
      You can use integer literal 10 as a decimal literal 10.
      a5’s type is float. This is inferred from the type checking algorithm.
      a6’s type is float. This is inferred from the type checking algorithm.
