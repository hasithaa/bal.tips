      String Literal With Escape Characters #

      There are certain code points, that are not allowed inside a string literal. You can use String Escape or Numeric Escape syntaxes to provide such code points.

      String Escape syntax #

      This syntax starts with \, then followed by one of t, n, r, \, " letters.

       1import ballerina/io;
 2
 3public function main() {
 4
 5  string tab = "tab :A\tB";
 6  string newline = "newline :A\nB";
 7  string carriageReturn = "carriageReturn :A\rB";
 8  string backSlash = "backSlash :A\\B";
 9  string doubleQuote = "doubleQuote :A\"B";
10
11// Some lines (i.e io:println()) are trimmed.
12// Open string-escape.bal to view them.
13}
