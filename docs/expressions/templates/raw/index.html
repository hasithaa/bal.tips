      Raw Template Expression #

      Raw Template expression constructs an object belongs to object:RawTemplate object type.

      • Syntax - e.g., `A ${a} B`
       1import ballerina/io;
 2
 3public function main() {
 4
 5  string name = "";
 6  int age = 30;
 7  object:RawTemplate tmp = `My name is ${name}. My age is ${age}.`;
 8
 9  io:println(tmp.strings);
10  io:println(tmp.insertions);
11}

      $ bal run raw-template.bal

["",30]
