JSON Compatibility

For JSON compatibility, you can also write () as null. This will help you to copy and paste JSON values as expressions in your code. However, avoid using null in places unrelated to JSON.

import ballerina/io;

public function main() {

  () b = null;
  null c = null; // Also valid
  io:println("a:", b);
  io:println("b:", c);

  // Using JSON value as expr
  json tom = {"name": "tom", "job": null};

  json bob = {"name": "bob", "job": ()};

  io:println("person:", tom);
  io:println("person:", bob);
}
