'Hello, World...!' and Ballerina Basics

Let us learn how to write a simple Ballerina program.

  • Ballerina is a C-style language.
  • Main function acts as an entry point for a program.
  • A Module is a collection of Ballerina codes.
  • Variable declaration
    type identifier = expr;
import ballerina/io;
public function main() {
io:println("Hello, World...!");
}

Comments

You can use comments to provide more details about the code or temporally disable a section of the source code.
  • //
  • No Multiline Comments.
  • Only for Comments.
  • Not for language docs nor metadata.

// io:println("No execution.");

Nil Data Type

The nil basic data type represents A single value called nil, which represents the absence of any other value.
  • ()
  • Nil Literal : () or null (for JSON support).
  • Multiple toString representations.

() a = ();
() b = null;
json tom = {"name": "tom", "job": null};

Boolean Data Type

The boolean basic data type represents boolean truth values: true and false.
  • boolean
  • Boolean Literal, e.g. true & false
  • Lang Library : ballerina/lang.boolean

boolean enable = true;
boolean open = false;
string s = true.toString();
boolean t1 = check boolean:fromString("true");
boolean t2 = check boolean:fromString("1");
boolean f1 = check boolean:fromString("false");
boolean f2 = check boolean:fromString("0");

Integer Data Type

The integer basic data type represents 64-bit signed integer values.
  • int
  • Integer Literal e.g. 10, 12345, 0xAD
  • Lang Library : ballerina/lang.int
  • byte and more subtypes.
  • No implicit conversion between numbers.

int i1 = 10;
int i2 = -5;
int i3 = i1 + i2;
int i4 = int:MAX_VALUE;
int i5 = int:MIN_VALUE;
int v1 = <int>floatValue;
int min = int:min(1, -2, 3, -4, 5);
int abs = min.abs();

Integer Literal

Integer values can be expressed in either common the Decimal system, known for its base 10 format, or the Hexadecimal system which uses the Base 16 format.
  • Decimal Literal, e.g. 123
  • Hexadecimal Literal, e.g. 0x1a2b
  • Conversion using to/from string functions.

  int i1 = 34368;
  int i2 = -287;
  int ih1 = 0X4D3B;
  int ih2 = -0x6e1a;
  string ts = 19467.toString();
  int fs = check int:fromString(ts);
  string hex = 19467.toHexString();
  int fhs = check int:fromHexString(hex);

Float Data Type

The float basic data type represents 64-bit IEEE 754-2008 binary floating point numbers.
  • float
  • Floating Point Literal, e.g. 12.34, 0xA1.2B
  • Lang Library : ballerina/lang.float
  • No implicit conversion between numbers.

Decimal Data Type

The decimal basic data type represents 128-bit IEEE 754-2008 decimal floating point values.
  • decimal
  • Floating Point Literal, e.g. 12.34,
  • Lang Library : ballerina/lang.decimal
  • No implicit conversion between numbers.

Array Type

An Array type describes a set of list values. A list value is a container that keeps its members in an ordered list and the index of the member is used to access an member. An Array type specifies a type, and all members of the list must belongs to that type. The Array type syntax is variable length (open) by default unless its length is specified.
  • Variable Length : 𝓣[]
  • Fixed Length : 𝓣[𝓲]
  • Fixed Length - Inferred : 𝓣[*]
  • Multi Dimension : 𝓣[][][], T[𝓲][𝓳][𝓴]

Here 𝓣 is the member type, 𝓲, 𝓳 and 𝓴 are integer values(or integer constants)

  • List Constructor
    e.g. [1, 2, 3, 4], [[1,2], [3,4]]
  • Lang Library : ballerina/lang.array
  • Iterable 🔁

Map Type

A map type describes mapping values. A mapping value is a container where each member has a unique string key, which is used to look up a member. A map type specifies a type, and all members of the mapping value must belong to that type.
  • map<𝓣>

Here 𝓣 is the member type.

  • Mapping Constructor
    e.g. {𝓪 : 1, 𝓫 : true}
    Here 𝓪 and 𝓫 are filed names.
  • Lang Library : ballerina/lang.map
  • Iterable 🔁

Record Type

Similar to map type, A Record type describes mapping values too. Similar to Tuple syntax, a Record type specify a separate type for each member of the mapping value. The term field is used to represent both the type of the member and its key together.
  • Open Record - Inclusive (Open to anydata)
    record { 𝓣 𝓪; 𝓢 𝓫; }
  • Open Record - Exclusive
    record {| 𝓣 𝓪; 𝓡...; |}
  • Close Record - (No Rest Field)
    record {| 𝓣 𝓪; 𝓢 𝓫; |}
  • Optional Field
    record {| 𝓣 𝓪?; |}
  • Readonly Field
    record {| readonly 𝓣 𝓪?; |}
  • Record Inclusion
    record {| *𝓣; |}

Here 𝓣 and 𝓢 are member types with their field names 𝓪 and 𝓫. 𝓡 is the rest member type.

  • Mapping Constructor
    e.g. {𝓪 : 1, 𝓫 : true}
    Here 𝓪 and 𝓫 are filed names.
  • Lang Library : ballerina/lang.map
  • Iterable 🔁

String Data Type

The string data type is a representation of a sequence of zero or more Unicode characters.
  • string
  • String Literal
    e.g. "Hello, World...!"
  • String Template
    e.g. string `Hello, ${name}...!`
  • Lang Library : ballerina/lang.string
  • Iterable 🔁

Table Type

The table Type represents a tabular data structure, where members are mapping values. The table keeps members in order, but unlike list values, you cannot access members using an index. Table type defines a key to access its members.
  • Table with Individual Keys
    table <𝓣> key (𝓪, 𝓫)
  • Table with Key Constrain 𝓢
    table <𝓣> key <𝓢>
  • Keyless Table
    table <𝓣> or table <𝓣> key <never>
  • Any Table Value
    table

Here 𝓣 and 𝓢 are Record types. 𝓪 and 𝓫 are table keys which are required and readonly field names with subtype of anydata of the 𝓣.

  • Table Constructor, e.g.
    table [{𝓪 : 1, 𝓫 : true}, {𝓪 : 2, 𝓫 : false}]
    Here 𝓪 and 𝓫 are filed names.
  • Lang Library : ballerina/lang.table
  • Iterable 🔁
