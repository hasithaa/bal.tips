Tuple Type

Like an Array, a Tuple type describes a set of list values. However, a Tuple type specifies a set of types separately for each member of the list. The Tuple type syntax is fixed in length by default, but it can be variable length with a rest field.
  • Fixed Length : [𝓣, 𝓢]
  • Open Length : [𝓣, 𝓡...]

Here 𝓣 and 𝓢 are member types. 𝓡 is the rest field type.

  • List Constructor
    e.g. [1, 2, 3, 4]
  • Lang Library : ballerina/lang.array
  • Iterable 🔁

XML Data Type

The xml data type is a representation of a sequence of parsed XML. It can contain various elements such as XML elements, processing instructions, comments, and text that appear within an XML element.
  • Any XML value : xml
  • Constraint XML Type : xml <𝓣>

Here 𝓣 is subtype of xml:Element|xml:ProcessingInstruction|xml:Comment|xml:Text

  • XML Template
    e.g. xml `<name id="12">${text}</name>`
  • Lang Library : ballerina/lang.xml
  • Iterable 🔁
