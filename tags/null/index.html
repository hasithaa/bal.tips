ðŸ’¡  Null Literal

For JSON compatibility you can also write () as null. This will help you to copy and paste JSON values as expressions in the code. But avoid using null in non-JSON-related places.

import ballerina/io;

public function main() {
  () a = null;
  null b = null; // Also valid

  io:println("a:", a);
  io:println("b:", b);

  // Using JSON value as expr
  json tom = 
   {"name" : "tom", "job" : null};

  json bob = 
   {"name" : "bob", "job" : ()};
  io:println("person:", tom);
  io:println("person:", bob);
}
bal run null-literal.bal
