Hello World

hello-world io:println

Writing a simple Hello, World is easy…! Let’s write something to stdout first.

import ballerina/io;

public function main() {
  io:println("Hello World...!");
}

How to run this example?

  • Download main.bal file.
  • Open the terminal and cd to the main.bal file location.
  • There are two ways to run this.
    • Execute it directly using bal run main.bal command.
    • Build the executable jar file first using bal build main.bal. Then run it using bal run main.jar

Hello World API

hello-world http:service

Let’s create a Hello, World HTTP API. This API will accept HTTP get request at http://localhost:8080/greeting. I will use ballerina/http module to implement this API.

import ballerina/http;

service on new http:Listener(8080) {
  resource function get greeting(string? name) returns json { 
    if name == () {
      return {"content": "Hello, World!"};
    }  
    string content = string `Hello, ${name}!`;
    return {"content": content};
  }
}
View Source (service.bal)

How to run this example?

  • Download service.bal file and start the program using bal run service.bal command.
  • Visit http://localhost:8080/greeting. You will see {"content":"Hello, World!"} JSON output.
  • Now let’s try providing a value to the name query parameter. Visit http://localhost:8080/greeting?name=Ballerina. You will see {"content":"Hello, Ballerina!"} JSON output.
